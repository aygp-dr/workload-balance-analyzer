# workload-balance-analyzer

Team workload visualization. Terminal UI built with [Bubble Tea](https://github.com/charmbracelet/bubbletea).

## Run

```bash
go run .
```

## Build

```bash
make build
./bin/workload-balance-analyzer
```

## Controls

- `j/k`: navigate
- `Enter`: select/details
- `q`: quit
